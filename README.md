# Privacy Policy for YouTube English Learner

**Last Updated:** April 13, 2026

## 1. Information We Collect
YouTube English Learner stores your saved vocabulary words and extension settings locally in your browser using `chrome.storage.local` and `localStorage`. We do not collect, transmit, or store your personal data on any remote servers.

## 2. Third-Party Services
This extension uses the following third-party APIs to provide core functionality:

- **Google Translate API** — Receives English subtitle text to return Chinese translations.
- **Free Dictionary API (dictionaryapi.dev)** — Receives clicked words to return definitions, phonetics, and example sentences.

These APIs only receive the specific text you interact with during normal use of the extension.

## 3. Data Sharing
We do not sell, trade, or share your personal data with any third parties.
